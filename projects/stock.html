Inventory Retail Power BI

Stock

Inventory control dashboard for a fictional clothing store chain. Built to monitor stock levels across all units and surface actions needed to maintain product availability and prevent revenue impact. A highlight of this project is the use of a hierarchical tree chart — an uncommon Power BI visual — for displaying item hierarchies across stores.

Key Features & Metrics

Hierarchical tree chart (custom visual)
Product filter by category
Number of products per store
Return percentage
Item quantities by unit
Drill-down by product
